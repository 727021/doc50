# Introduction

Welcome to Doc50! This is a collection of unofficial documentation for tools and topics related to CS50 courses.

<Callout type="info">
  Not sure what CS50 is? Check it out [here](https://cs50.harvard.edu/).
</Callout>

This site is not endorsed by or affiliated with CS50, but is intended to be a helpful resource for students and answer common questions.
This documentation is a work in progress, and we welcome contributions! If you have suggestions for improvements or new topics to cover,
please [open an issue](https://github.com/727021/doc50/issues) or [submit a pull request](https://github.com/727021/doc50/pulls) on our GitHub repository.

This site is *not* for getting help with specific CS50 assignments or projects. If you need help with a specific assignment, please ask for help in one of [CS50's communities](https://cs50.harvard.edu/communities/).
